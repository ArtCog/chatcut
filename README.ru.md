# chatcut

> Одна команда превращает сырое talking-head видео в готовый ролик — через твоего агента (Claude Code / Codex), без таймлайна.

**chatcut** — расширяемая агентная видео-студия для **talking-head** контента (влоги, туториалы, объяснялки). Кидаешь сырьё, говоришь словами что нужно — агент гонит боевой пайплайн: транскрипт → убрать паузы и оговорки → субтитры → моушн-графика → цвет → рендер.

Он **не переизобретает** движки. Он оркестрирует лучшие открытые инструменты — [video-use](https://github.com/browser-use/video-use), [hyperframes](https://github.com/heygen-com/hyperframes), [auto-editor](https://github.com/WyattBlue/auto-editor), [faster-whisper](https://github.com/SYSTRAN/faster-whisper), ffmpeg — и добавляет то, что реально сложно: **выстраданные правила корректности**, чтобы «одна команда» не выдавала битый ролик (нет зависших кадров, нет рассинхрона, нет убитого звука).

> 🚧 **Статус: v0.1 в активной разработке.** Пока не готово. Делается открыто.

## Зачем chatcut

- **Одна команда, а не copy-paste.** Настоящий CLI + агент-оркестрация вместо вставки промптов в IDE.
- **Глубина в talking-head.** Нарезка по смыслу, чистка филлеров, ловля оговорок, субтитры в синхрон со словами.
- **Бесплатно по умолчанию.** Локальный `faster-whisper` из коробки — без платного ключа. Премиум-движки (ElevenLabs Scribe) — опциональный апгрейд.
- **Кроссплатформенный энкод.** NVENC если есть, иначе fallback на libx264 / VideoToolbox / QSV.
- **Создан расти.** Реестр плагинов — новые тулзы и пайплайны добавляются без правок ядра.

## Как работает (talking-head пайплайн)

```
сырьё
  → нормализация (чистый CFR)         не рассинхронить на VFR-входе
  → транскрипт (word-level)           faster-whisper (локально) / ElevenLabs (опц.)
  → детект оговорок и пауз
  → cut-plan (ты подтверждаешь)        никогда не режет вслепую
  → нарезка (EDL, padding по типу)
  → субтитры
  → моушн-графика (опц.)               hyperframes
  → цвет (LUT)
  → рендер                            результат с проверкой корректности
```

## Статус и рост

v0.1 — рабочий talking-head пайплайн за одной командой. Рост (shorts, подкаст, B-roll, диаризация, денойз, замена фона) — плагинами.
